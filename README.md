# PowerBI_FinalPr
# Final Project – Sales & Customer Intelligence Dashboard

## Project Overview
This Power BI project provides insights into sales performance, customer behavior, returns, and regional trends using interactive dashboards and DAX measures.

The report was designed to support business decision-making through KPIs, time intelligence, filtering, and drillthrough analysis.

---

## Objectives
- Build a Star Schema data model
- Create DAX measures and calculated columns
- Apply Time Intelligence (YTD, YOY, MOM)
- Develop interactive dashboards
- Add slicers, drillthrough, and navigation features
- Optimize mobile layout
